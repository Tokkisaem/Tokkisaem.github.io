HTML, CSS, JS

간단히 설명하자면, HTML은 뼈대를 잡아주고, CSS는 디자인을 해주고, JavaScript는 동작을 처리해주는 역할을 합니다!

HTML로는 제목이나, 글자 또는 이미지, 영상 등의 요소 (태그) 를 정의합니다. 웹의 구조를 잡아준다고 봐도 괜찮을 것 같아요!

그리고 CSS는, HTML에서 정의해준 요소들을 꾸며줍니다! 색깔이나 간격, 크기, 폰트 등.. 시각적 표현을 담당해줍니다.

JavaScript로는, 버튼을 클릭하거나, 컨텐츠를 바꾸고 움직이는 등.. 웹 페이지에서의 동작들을 처리하는 담당을 합니다!

이 3개의 언어는 밀접한 관계를 가지고, 웹 페이지를 구성하는 기본적인 언어들입니다!
