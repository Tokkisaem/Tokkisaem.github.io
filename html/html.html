HTML

이번엔 HTML에 대해 알려드리겠습니다!

HTML이라는 언어는, 구조를 만드는 역할을 하니 문법정도와 각 태그들의 역할, 어떤 속성을 사용하는지 정도만 아시면 될 것 같습니다 :)

HTML의 기본 문법

먼저, HTML의 문법을 알아봐요!

[일반적인 태그]

HTML은 '태그' 중심의 마크업 언어이기 때문에, 태그의 문법을 잘 알아야 합니다!

<태그> 내용 </태그>

태그들은 대부분 이러한 형태를 갖습니다.

앞쪽에 있는 태그를 시작(열린) 태그라 부르고, 뒤쪽의 태그를 종료(닫힌) 태그라 부르죠. 그 사이에는 내용이 들어갑니다!

그냥 이건, 태그라는 새로운 개념이라고 생각하시면 됩니다! 뒤에서 각각 태그들의 쓰임새를 알면 이해될거에요!

제가 앞에서, '대부분' 이러한 형태를 가진다고 했죠? 이런 형태를 가지지 않는 태그들도 있습니다.

[빈 태그]

<태그 />

이러한 형태를 가진 태그는, '빈 태그' 라고 부르고 닫히는 태그가 존재하지 않습니다.

그리고 이러한 빈 태그는, 닫히는 태그 대신에 열린 태그의 마지막에 /를 붙여줍니다!

대표적으로는 이미지를 넣어주는 img 태그, 줄바꿈을 해주는 br 태그가 있습니다.

[태그의 속성]

<태그 속성="값"> 내용 </태그>

태그마다 각각의 속성을 적용하여 기능을 줄 수도 있습니다!

이미지 태그를 예로 들어볼까요?

<img />

지금은 그냥.. 속성이 없어서 이미지 태그만 있고 어떤 이미지를 보여줄지 모르겠죠..?

<img src="link" alt="text" />

하지만 이런식으로 속성에 값을 주어 기능을 추가하면, 더 많은 일을 할 수 있게 되는거에요!

아아, '태그' 라는 말은 '요소' 라는 말로도 이해 할 수 있습니다!

혹시 어려운..가요..? HTML은 이것만 알면 끝이니 안심하세요!!

HTML의 기본 구조

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
</head>
<body>
</body>
</html>

일단, HTML의 기본 구조부터 알아볼까요?

이 구조는, 꼭 필요한 필수구조입니다. 순서대로 설명해볼게요!

[!DOCTYPE html]

!DOCTYPE html 태그는, 이 웹 페이지의 HTML 버전을 지정해주는거에요. 최신버전 - HTML5로 지정하는거죠!

이 태그는 특수한 태그라 빈 태그 취급도 받지 않고, 저렇게 열린 태그만 존재합니다.

[html]

html 태그는 문서의 전체 범위를 브라우저에게 알려주는겁니다!

브라우저도 이 태그를 통해 아, 여기가 html의 범위구나? 하고 알아채는거죠.

[head]

head 태그는 문서의 정보를 나타내주는 범위입니다.

사람들에게 보여지지 않는 정보들, 페이지의 제목이나 설명, 가져올 파일의 위치 등등.. 을 쓰는 범위입니다!

[body]

body 태그는 문서의 구조를 나타내주는 범위입니다.

사람들에게 보여지는, 로고라던가 메뉴.. 버튼.. 글, 제목 등등을 쓰는 범위에요!

TMI

사실 여기까지는 다 알아야 하지만.. 앞으로 배울 HTML, CSS는 반복해서 써보면 깊게 알지 못해도 괜찮아요.

저도 기억이 나지 않을 때는 이렇게 검색 치트키를 (?) 사용하니까.. 모르면 검색하시면서 하셔도 되니 기본만 잘 배워봐요!

그리고 HTML과 CSS는, 이렇게 기본을 열심히 배우고 많이 써보는게 중요해요!!

이 아래에 있는 더 알아보기는.. 정말 중요한 내용이라고 생각해요. 꼭 한번씩 보길 바래요 :)

head 안의 태그

이번엔, head 태그 안에 들어가는 태그들을 알아봅시다!

[title]

<title> 탭 제목 </title>

처음으로는 title 태그인데요. 웹 브라우저의 탭을 아시나요?

바로 이 화면에 들어갈 글씨를 넣어주는 태그입니다!

[link]

<link href="파일의 링크" rel="불러올 파일의 쓰임새" />

뭔가 길죠..? 하지만 이 link 태그는, 거의 두가지 용도로밖에 사용하지 않으니 잘 알아두면 좋습니다!

css 파일을 가져올 때와, title 태그로 지정 해 준 제목 옆에 들어갈 아이콘을 설정 할 때 쓰입니다.

<link href="링크" rel="stylesheet / icon" />

rel 속성에 아이콘을 지정 할 때는 icon, css 파일을 가져올 때는 stylesheet 라는 값을 주시면 됩니다! 쉽죠?

[style]

<style> css 코드 <style/>

이건 나중에 css를 배운 후에 이 안에 css 코드를 적는 용도입니다.

[meta]

<meta charset="utf-8" />

meta 태그로는 문자의 인코딩 방식을 지정하여서 모든 문자들이 제대로 보이게 설정할 수 있습니다.

이와 같은 용도 말고도, 페이지의 키워드나 저자.. 페이지의 확대 여부, 배수 등등을 설정 할 수 있는데..

HTML - Plus Ver에서 확인해주세요!

body 안의 태그

이번에는 body 태그 안에 들어가는 태그들을 알아봐요!

[h1~6]

<h1~6> 제목 태그 </h1~6>

먼저, 1~6까지의 숫자의 의미는 글자의 크기입니다! 1이 가장 크고 중요하고, 6으로 갈수록 작아지고 중요도가 떨어지는 의미를 지닙니다.

그리고 h라는 영어의 의미는 제목, Heading에서 따온 의미입니다 :)

[p]

<p> 본문, 문장 </p>

p 태그는 문장이나 본문을 의미하는 태그입니다! 그냥 글을 쓸 떄, 이 태그를 쓰시면 될 것 같아요! 이 태그를 쓰면 줄바꿈도 된답니다.

p라는 영어는 Paragraph에서 따왔다네요!

중간의 이상한 영어들은 특수문자 표기이니 무시하시고, 대충 이런식으로 쓰이는 태그들입니다! 앗, 혹시 특수문자 표기가 궁금하신가요?

더 알아보기 : 특수문자 표기

[img]

<img src="이미지 링크" alt="대체 문자" />

이번에는 이미지를 넣는 img 태그입니다! src 속성에는 이미지의 링크, alt 속성에는 이미지가 보이지 않을 때 나올 문자를 적어주시면 됩니다.

당연하게도 img는 Image라는 영어에서 따온 의미라네요 :)

[ul, ol, li]

<ul / ol>
<li> 항목 이름 </li>
</ul / ol>

ul 태그는 순서가 필요 없는 목록의 집합 (Unordered List) / ol 태그는 순서가 필요 있는 목록의 집합 (Ordered List)의 의미를 가지고,

li 태그는 목록 내의 항목의 의미 (List Item)를 지닌다는데.. 그냥 한번 보시면 이해가 잘 될 것 같아요!

ol 태그에는 type, start 등의 속성이 있어서 다양하게 사용할 수 있답니다. 앗, 이것도 알아볼까요?

더 알아보기 : 태그 보충 설명

[a]

<a href="링크" target="탭 열리는 방식"> 내용 </a>

a 태그는, 가운데 내용을 누르면 href 속서의 링크로 이동하는 태그입니다!

href 속성에 #name 을 이용하여 같은 페이지 안에서도 이동 할 수 있는데, 예시를 보여드리겠습니다!

이동하고자 할 곳의 a태그에 name이라는 속성에 값을 주고, 누르면 이동 할 a 태그의 href 속성에는 name이라는 속성에 준 값 앞에 #를 붙여주면 됩니다!

target 속성은 두가지로 쓰이는데요. 기본값인 _self는 생략이 가능하고, 클릭하면 문서를 현재 탭에서 엽니다!

그리고 _blank 라는 값을 주면, 새 탭에서 창을 열게 되죠.

네이버로 이동 (target="_self") / 네이버로 이동 (target="_blank")

[table, tr, th, td]

<table border="1">
<tr>
<th rowspan="2">표 내용1</th>
<th>표 내용2</th>
<th>표 내용3</th>
</tr>
<tr>
<td colspan="2" align="center">표 내용4</td>
</tr>
</table>

table, tr, th, td는 표를 만들 때 유용하게 쓰이는 태그인데요. 바로 예시로 알아보죠!

먼저 표를 만들 때, table은 표 전체 구역을 감싸주는 역할을 합니다.

그리고 tr (Table Row) 은 행 (세로) 을 지정해주는 역할을 해서, 예시를 보면 tr이 두번 나왔으니 2줄이 만들어져 있는게 보이시나요?

th와 td (Table Data) 는 열 (가로, 셀) 을 지정해주는 역할을 합니다.

그런데, 이상한 속성들이 있죠? 저것도 알려드리겠습니다!

table 태그에는 border라는 속성이 붙을 수 있는데, 숫자를 크게 할수록 표의 테두리가 두꺼워지는 성질을 가집니다. (기본값 : 0)

td, th에는 rowspan / colspan이라는 속성을 쓸 수 있는데.. 혹시 병합이라는 말 들어보셨나요?

rowspan은 세로축으로 값만큼, colspan은 가로축으로 값만큼 셀을 합칩니다!

마지막으로 align 속성은 문자를 정렬해줍니다! 값으로는 left (기본), center, right 등이 들어갑니다. 사용이 권장되지 않으니 나중에 css로 정렬하는걸 추천드립니다.

[div, span]

<div>내용</div>
<span>내용</span>

이번엔 div (Division) 와 span입니다!

div 태그는 구역을 나누기 위해서 사용하고, span 태그는 의미 없이 구분을 하기 위해 사용합니다!

둘 다 특별한 의미는 없이 구역을 나누는 태그들이지만, css로 가면 쓸 일이 많아질거에요! (태그들을 선택하여 스타일을 주어야 하기 때문에)

[br]

<br />

마지막으로 br (Break) 태그입니다!

이 태그를 쓰게 되면, 줄 바꿈을 해줍니다!

p 태그나 h태그에 줄바꿈이 기본적으로 있기는 하지만, html만 가지고 여러 줄을 바꿀 때 사용합니다.
